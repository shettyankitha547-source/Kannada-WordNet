# Kannada WordNet Interface

## 📖 Project Overview

A user-friendly interface for exploring **Kannada WordNet** using Python and Google Colab. The project improves the usability of IndoWordNet by providing a clean interface with lexical search and interactive features.

## ✨ Features

- 🔍 Kannada Word Search
- 📖 Gloss (Meaning)
- 📝 Example Sentences
- 🔗 Synonyms
- 🏷️ Part of Speech (POS)
- 🆔 Synset ID
- ⌨️ Virtual Kannada Keyboard
- 🌙 Dark Mode
- 💡 Search Suggestions
- 📂 CSV Dataset Integration

## 🛠️ Technologies Used

- Python
- Google Colab
- Pandas
- HTML
- CSS
- JavaScript
- IndoWordNet

## How to Run

1. Open `notebooks/indo_interface.ipynb` in Google Colab.
2. Upload the dataset (`word.csv` or `words.xlsx`).
3. Run all notebook cells.
4. Use the interface to search Kannada words.
